## Simple NodeJS blog application using JWT for authentication

### How to start the app locally:

- `git clone https://github.com/MWronski12/my-awesome-blog`
- `cd backend`
- `npm i`
- `npm run dev`
- Open second terminal in the project root directory
- `cd frontend`
- `npm i`
- `npm run dev`
